Sudoku generator API v.1.

Sudoku generator API will allow you to get your daily sudoku grid and even more.

See for yourself!

Sudoku generator API v.1 details

  1. Generate new sudoku

    {{base_url}}api/gen/
    Request type: POST with board_diffic='DIFFICULTY'.

    Generates a brand new sudoku grid with selected DIFFICULTY.

    Try this example: {{base_url}}api/gen/ {%csrf_token%}

    DIFFICULTY values:

    • 0 - uber easy sudoku (baby)
    • 1 - easy sudoku
    • 2 - medium sudoku
    • 3 - hard sudoku

  2. Get last generated sudoku boards

    {{base_url}}api/last/
    Request type: GET

    Returns 25 last generated sudoku grids.

    Try this example: {{base_url}}api/last

  3. Get sudoku board solution

    {{base_url}}api/check?pk=SUDOKUBOARD_ID
    Request type: GET

    Takes sugoku board ID and returns sudoku board and the solution for it.

    Try this example: {{base_url}}api/check?id=BOARD_ID (you'll need to replace BOARD_ID with existing one)
