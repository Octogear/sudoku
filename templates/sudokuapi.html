Sudoku generator API v.1 details

  1. POST 

    {{base_url}}api/v1/gen/DIFFICULTY
    generates a brand new sudoku grid with selected DIFFICULTY.

    DIFFICULTY values:

    • 0 - sudoku for kids
    • 1 - easy sudoku
    • 2 - medium sudoku
    • 3 - hard sudoku

  2. GET 

    {{base_url}}api/v1/last/
    returns 25 last generated sudoku grids.
